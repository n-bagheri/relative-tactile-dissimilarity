# Stimuli

This folder contains the experimental stimuli used in the study reported in the paper:

**Toward a Relative Metric for Tactile Dissimilarity: Empirical Insights for Modeling Transition Saliency in Tactile Graphics**

## Overview

The materials in this folder were used as part of the experimental protocol investigating tactile dissimilarity and transition saliency in tactile graphics.

## Contents

This folder includes the stimuli presented in the experiment.
